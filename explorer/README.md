# Hyperledger Explorer - Quick Start

## Khởi Động Explorer

```bash
cd /opt/fabric/fabric-real-estate-network/network
./explorer-up.sh
```

## Truy Cập Explorer

- URL: http://localhost:8080
- Username: `exploreradmin`
- Password: `exploreradminpw`

## Dừng Explorer

```bash
cd /opt/fabric/fabric-real-estate-network/network
./explorer-down.sh
```

## Xem Logs

```bash
docker logs -f explorer.mynetwork.com
```

## Tài Liệu Đầy Đủ

Xem file `EXPLORER_SETUP_GUIDE.md` để biết hướng dẫn chi tiết về cài đặt, cấu hình và troubleshooting.
